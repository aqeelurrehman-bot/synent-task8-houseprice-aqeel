# Synent Internship Task 8 - House Price Prediction

 Problem Statement
The goal of this project is to build a machine learning model that predicts house prices using housing data.

 Dataset Used
California Housing Dataset / housing.csv

 Objective
- Perform data preprocessing  
- Train machine learning model  
- Predict house prices  
- Evaluate model performance  

 Tools Used
- Python  
- Pandas  
- Scikit-learn  
- Visual Studio Code  

 Steps Performed
1. Loaded the dataset  
2. Handled missing values  
3. Encoded categorical columns  
4. Split data into training and testing sets  
5. Trained Linear Regression model  
6. Predicted house prices  
7. Evaluated using RMSE and R2 Score  

 Results
A working machine learning model was successfully created for house price prediction.
